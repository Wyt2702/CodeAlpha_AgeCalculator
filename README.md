# CodeAlpha Age Calculator

This project is a simple web-based **Age Calculator** developed using **HTML, CSS, and JavaScript**.

The application allows users to enter their **date of birth** and calculates their **age in years, months, and days** instantly.

## Features
- User-friendly interface
- Date of birth input
- Accurate age calculation
- Displays age in years, months, and days

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Used
- DOM Manipulation
- JavaScript Date Object
- Event Handling
- Input Validation

## Project Purpose
This project was created as part of the **CodeAlpha Web Development Internship** to practice building interactive web applications using basic front-end technologies.
